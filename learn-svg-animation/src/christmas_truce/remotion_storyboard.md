# Christmas Truce - Remotion SVG Storyboard

This storyboard is highly optimized for 2D SVG programmatic animation in Remotion. It adheres to the rules of **one action per scene**, **limited riggable characters (1-2 max)**, and **2D camera movements**.

### Global Cinematic Filters (Applied to all scenes)
- **Color Grading:** `filter: "contrast(1.1) sepia(0.2) saturate(0.85)"` applied to the main composition to give a gritty, vintage World War 1 movie look.
- **Vignette:** A dark inner shadow (`box-shadow: inset 0 0 150px rgba(0,0,0,0.8)`) around the edges of the screen to draw focus to the center.
